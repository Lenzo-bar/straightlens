# StraightLens

StraightLens is a rich, values-guided SaaS knowledge platform that turns user questions into structured answers with practical next steps, visual sections, and compliance-aware moderation.

## Stack
- Frontend: Next.js 14 + Tailwind
- API: FastAPI
- Database: PostgreSQL
- Email: Resend
- Hosting: Vercel + Render + Neon

## Core capabilities
- Ask-anything interface
- Structured answers
- Safety and moderation notes
- User accounts
- History, retention, deletion controls
- Admin-grade audit trail
